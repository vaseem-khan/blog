Untrack Files From Git

Fri 26 December 2014 | 1 min read | By Vaseem Ahmed Khan

When devloping we encounter that a file is tracked under git, you have made changes to it but now needs to push.

One example can be changes in setings(local changes) file.

One option is to stash changes. Push and apply stash. Follow these steps at time of every push.

but I prefer using git update-index

git update-index --assume-unchanged file_name

Now this file will be untrack until you want to start keeping track again

By excuting thois command

git update-index --no-assume-unchanged file_name

and we escape from frequent stash before every push.
